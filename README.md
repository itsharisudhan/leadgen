# LeadGen 🚀

**Find businesses that need your services — before anyone else.**

LeadGen helps freelancers discover local businesses without websites or social media, so you can pitch them with a ready-made proposal.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the landing page.

## Tech Stack

- **Next.js 15** (App Router + Turbopack)
- **React 19**
- **Tailwind CSS 4**
- **TypeScript**

## Deploy

Push to GitHub and connect to [Vercel](https://vercel.com) for instant deployment.
